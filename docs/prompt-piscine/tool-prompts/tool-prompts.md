# tool-prompts 

## Overview

- Using language models with external tools (calculator, search, API)  
- Basics of function calling for structured responses

## Learning Objectives

- Understand prompt chaining and tool use  
- Design prompts that request structured JSON output  
- Simulate API calls with controlled prompts  
- Combine multiple prompts to mimic simple agent flows  

## Exercises of the quest

> **Note:** All submitted answers will be scanned using an AI Detector. If your response is identified as being generated by an AI tool (ChatGPT, etc.), it will be rejected. Make sure to write answers in your own words and avoid copying AI-generated text.

- Exercise 0: Environment and Libraries  
- Exercise 1: Structured JSON output  
- Exercise 2: Simulated API call (weather example)  
- Exercise 3: Chaining prompts for agent flow  

## Virtual Environment

- Playground or chat interface for testing prompts  
- Python 3.9+ installed locally (optional, for simulating API workflows)  

## Exercise 0: Environment and Libraries

**Objective:** Prepare the environment for work.

- Install Python (latest stable version, 3.9 or above).  
- Verify installation with:

 ```bash
 python --version
```

* Write and run a Hello, World! program.

## Exercise 1: Structured JSON Output

**Objective:** Create prompts that produce structured machine-readable output.

* Write a prompt that asks for unstructured information (e.g., “List three cities with their populations.”).

* Rewrite the prompt to enforce structured JSON output:

  > Example: “Return the following in JSON format: [{city: string, population: number}].”

* Test with multiple inputs and verify the JSON stays valid.

## Exercise 2: Simulated API Call (Weather Example)

**Objective:** Practice designing prompts that mimic function calls.

* Create a prompt: “Simulate a weather API response for Paris today.”

* Define an expected JSON schema, e.g.:

```json
 {
    "location": "Paris",
    "temperature_celsius": 18,
    "condition": "Cloudy"
 }
```

* Test the model with different cities and compare outputs.

## Exercise 3: Chaining Prompts for Agent Flow

**Objective:** Combine multiple prompts to simulate a tool-using agent.

* Step 1 prompt: Ask for a user’s query (e.g., “What’s the temperature in Paris?”).
* Step 2 prompt: Simulate a call to the weather API.
* Step 3 prompt: Format the final answer back to the user.
* Document the chained process and outputs.

## Submission

Provide all exercises. For each exercise include the prompt, outputs, and a brief sentence analysis demonstrating how prompts were refined or how the workflow produced structured results. For Exercises 1–3, show comparisons (weak vs revised prompts, JSON validation, or chained steps) to illustrate improvements or reasoning. Ensure all outputs are structured, consistent, and clearly documented.
